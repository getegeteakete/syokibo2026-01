# 100億宣言 申請サポートシステム

中小企業成長加速化補助金（第2次公募）の申請を支援するReactアプリケーション。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `100oku-unified.jsx` | **統合版（推奨）** ログイン・申請管理・制度説明ガイドを1つに統合。ヘッダーメニューで画面切替 |
| `100oku-secure-dashboard.jsx` | 申請管理ダッシュボード単体（セキュリティ機能付き） |
| `100oku-guide.jsx` | 制度説明ガイド単体 |

## 主な機能

### 認証・セキュリティ
- ハッシュ認証によるログイン（申請者 / サポートの2ロール）
- 30分セッションタイムアウト
- 5回連続失敗で60秒ロックアウト
- 機密項目のマスク表示
- 操作監査ログ

### 申請管理画面
- スケジュール管理（タイムライン表示）
- チェックリスト（進捗率表示）
- ヒアリングシート（5セクション・20+項目）
- 要件チェック＆申請入力（4ステップ・40+フィールド）
- AI申請書生成プロンプト
- サポートノート（サポート担当のみ）

### 制度説明ガイド
- 11セクション構成の完全ガイド
- サイドバー目次（スクロール追従）
- アコーディオンFAQ
- 千葉県の相談窓口情報

## デモアカウント

| ロール | ID | パスワード |
|--------|-----|-----------|
| 申請者 | `applicant` | `apply2026` |
| サポート | `support` | `support2026` |

## 技術スタック
- React (JSX)
- Noto Sans JP
- インラインCSS（外部依存なし）

## 対象制度
- 100億企業成長支援プロジェクト（100億宣言）
- 中小企業成長加速化補助金 第2次公募
- 締切：2026年3月26日 15:00
